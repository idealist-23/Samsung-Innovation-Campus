# Lab 2: Personal Branding Website with Vibe Coding

## Project Overview
This lab demonstrates an end-to-end "Vibe Coding" workflow using Generative AI. Without writing manual code, I directed the AI to define my professional profile, conceptualize visual assets, and generate a fully functional HTML/CSS personal branding website.
